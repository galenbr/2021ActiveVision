Active vision project

Run "roslaunch active_vision workspace.launch" to start gazebo with table, kinect and a object

For code verification purpose following functions are developed.
1) testKinectMovement function : Moves the kinect to the user input location
2) testKinectRead function : Reads the kinect data and displays it. Also, data on user input pixel is printed
3) testPtCldFuse function : Move kinect to 4 locations around the table and fuses the point cloud
